I made a simple web app that converts HTML, CSS, and js to Elementor structure that can be directly pasted to elementor.
Introducing... <Code2Elementor>
p/s: Once finished, this tool will be completely free for the elementor community.

Here are some current features I've included;
1) Write HTML, CSS and js, and automatically convert the code to elementor structure that can be pasted into the builder.
2) Once the code has been converted to elementor structure, it automatically gets named with the elements class or id. This can also be renamed, meaning the layer name will also be reflected in elementor once pasted.
3) In the layer panel, the layers can be renamed, reordered, deleted, duplicated or copy the specific layer and then it can be pasted in elementor. Actions such as ordering and deleting the layer will be reflected in the code, meaning if deleted, the corresponding element will be deleted in the HTML editor.
4) I've also included the ability to select elements in the canvas, and the corresponding element will be selected in the layer panel, this is great for inspecting the layout.
5) Before copying to elementor, one can decide if they want to apply the classes to elementor global classes, or add the classes to the element, inputed in CSS tab CSS classes.
6) Also, before copying to elementor, one can also decide to apply the CSS properties to elementor GUI interface, or apply the CSS properties in the custom CSS, with the classes tied to the CSS properties.
Basically, this is a fully fledged visual ide...made specifically for elementor.
p/s: The design in the image was made by ai in less than 2 minutes, ready to be pasted in elementor...I only did some polishing to match the variables with acss.
With all that said, this tool is not for complete beginners, in order to smoothly include it in your workflow, understanding the basics of HTML and CSS is crucial, especially since pasting in elementor, the layout may look different, this is because elementor has it global CSS rules, eg, all divs have max-width of 100%, which is some cases, may break layouts pasted to elementor directly. Nevertheless, I think with this tool, a lot of possibilities will be unlocked.
